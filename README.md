```
ZK_HOSTS
ZK_MYID
DLM_ADMIN_PASSWORD
```

```
BROKER_ID
ZOOKEEPER_CONNECT=<your_zookeeper_host_1>:2181,<your_zookeeper_host_2>:2181,<your_zookeeper_host_3>:2181
SCRAM_ADMIN_PASSWORD
SCRAM_BROKER_PASSWORD
SCRAM_CLIENT_PASSWORD
```
